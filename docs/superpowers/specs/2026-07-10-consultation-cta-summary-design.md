# Consultation Wizard CTA and Summary Panel Design

**Date:** 2026-07-10
**Status:** Approved

## Goal

Make the available-day action in the Player Consultation LatePoint wizard more obvious by changing its visible label from `Request this day` to `BOOK A CONSULTATION`, centering it, and giving it rounded, brand-forward button styling. Remove the LatePoint Summary panel from the consultation modal because consultation costs are not displayed and the panel is no longer useful to families.

## Scope

The change is limited to the modal-only Player Consultation wizard generated by the GV Members integration. Existing internal booking data, nominal start times, availability rules, venue selection, form fields, submission behavior, and coach finalization flow remain unchanged.

## Design

### Trigger configuration

Add LatePoint's native `hide_summary="yes"` attribute to every hidden consultation trigger emitted by `gv_members_hidden_booking_trigger()`:

- one trigger for each active venue;
- the `any`-venue trigger; and
- the fallback trigger used when no active locations are available.

The existing `hide_side_panel="yes"` attribute remains in place. Using the native LatePoint option allows the plugin to remove the panel and recalculate the form layout instead of depending on a fragile selector-only override.

### Visible day action

Keep the submitted `data-minutes` and `start_time` values untouched. In the existing post-render UI pass, relabel the visible `.dp-label-time` element inside each available `.timeslots .dp-timebox` to `BOOK A CONSULTATION` and add a namespaced class for styling. Any nominal compact time tick remains hidden as it is today.

The selected-slot cleanup remains limited to removing the nominal time from date summary text. Since the native Summary panel is hidden, no new summary replacement UI is introduced.

### Styling

Add scoped styles for the new action class under the existing LatePoint/consultation asset stylesheet:

- center the label in its day cell;
- use GV orange as the action background and white as the text color;
- use bold, readable typography and sufficient padding/minimum height;
- round the edges;
- provide a clear hover/focus treatment;
- preserve responsive sizing and the existing visible focus outline.

No LatePoint plugin files or generated WordPress content are edited.

## Verification

Add framework-free contract assertions to the existing GV Members contract test for:

- all consultation trigger shortcode variants containing `hide_summary="yes"`;
- the exact `BOOK A CONSULTATION` visible label and namespaced styling class in the JS source; and
- the CSS rules that center and round the new action.

Run PHP syntax checks, the affected contract test, and the full framework-free GV Members test suite. Review the final diff to confirm no unrelated working-tree changes are staged.

## Out of scope

- changing service duration, availability, or submitted appointment timestamps;
- removing the Members portal's own tabs or booking history summaries;
- changing costs or payment settings;
- redesigning the venue chooser or contact fields;
- modifying LatePoint core files.
